How to get an API Token
  1. Go to id.atlassian.com → Security → API tokens
  2. Click Create API token
  3. Give it a name (e.g. "JiraMaster")
  4. Copy the token and paste it above

The token is stored in the OS keyring when available (macOS Keychain, Windows Credential Manager, etc.), otherwise in config.json in the project directory. That file is gitignored and never committed. See the Security section in Status for the current mode.
